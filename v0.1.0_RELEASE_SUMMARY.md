## 🔧 Technical Details

### Settings Dialog Architecture
```python
SettingsDialog (QDialog)
├── LogSignals (QObject) - Signal emitter
├── QTextEditHandler (logging.Handler) - Log capture
├── Settings Tab
│   ├── Application Settings
│   ├── Data & Logging
│   └── Stock Viewer Settings
└── Logs Tab
    ├── Log Display
    ├── Log Filtering
    └── Export/Clear Functions
```

### Logging Flow
```
Application Code
    ↓ logger.info/debug/warning/error()
    ↓
QTextEditHandler (custom logging handler)
    ↓ emit signals
    ↓
LogSignals.new_log (PyQt6 signal)
    ↓ connected slot
    ↓
SettingsDialog._on_new_log()
    ↓ update display
    ↓
Live Log Viewer Display
```

### Build Process Enhancement
```
User runs build_executable.py
    ↓
Check dependencies
    ↓
Clean previous builds (optional)
    ↓
Run PyInstaller with spec file
    ↓
Generate checksums (SHA256)
    ↓
Save build metadata (JSON)
    ↓
Create build report
    ↓
Display summary
```
